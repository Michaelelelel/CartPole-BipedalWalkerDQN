\begin{abstract}

A bipedal walker is a system designed to move using two legs. In robotics and AI, it is often framed as a classical control or machine learning control problem. A DQN architecture capable of enabling the Gymnasium (formerly OpenAI Gym) bipedal agent to locomote across its environment has been devised, trained and evaluated.

\end{abstract}
